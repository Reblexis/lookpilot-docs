## LookPilot Configuration
1. Set protocol to `freetrack`
2. Click **Start** to begin tracking

## IL-2 Korea Setup
IL-2 Korea is built on the IL-2 Great Battles engine and reads head tracking through the freetrack protocol. Head tracking should work automatically once LookPilot is running. If the view does not respond, check the game's settings for a TrackIR or head tracking option that may need to be enabled.

IL-2 Korea is in Early Access, so tracking behavior may change between updates.
