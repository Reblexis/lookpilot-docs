## LookPilot Configuration
1. Set protocol to `auto`
2. Launch the game
3. Select the GTR FIA GT Racing Game Wine prefix in the `Game's Wine prefix` dropdown
4. Click **Start tracking** to begin tracking

### Fallback Option
If the above method doesn't work, try using the freetrack (Wine) protocol:

1. Set protocol to `freetrack (Wine)`
2. Select the GTR FIA GT Racing Game Wine prefix in the `Game's Wine prefix` dropdown
3. Launch the game
4. Click **Start tracking** to begin tracking

## GTR FIA GT Racing Game Setup
Once LookPilot is running and configured, head tracking should work automatically. If not, check the game's control or camera settings for TrackIR or head tracking options.
