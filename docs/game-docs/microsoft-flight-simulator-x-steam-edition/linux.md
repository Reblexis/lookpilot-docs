## LookPilot Configuration

1. Set protocol to `virtual joystick`
2. Map axes in **Tracking settings** (see [Virtual joystick](../../app-docs/connect-to-any-game/linux.md#2-virtual-joystick)).
3. Launch **Microsoft Flight Simulator X: Steam Edition** from Steam
4. Click **Start tracking**

## Microsoft Flight Simulator X: Steam Edition Setup

1. In **Settings → Controls**, bind pan and tilt (or cockpit view) to the virtual joystick axes. Prefer absolute (non-relative) control where available.

For SimConnect on Windows, see the [Windows guide](windows.md).
