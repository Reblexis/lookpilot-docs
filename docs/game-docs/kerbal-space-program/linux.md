## LookPilot Configuration
1. Set protocol to `freetrack`
2. Click **Start** to begin tracking

## Kerbal Space Program Setup
KSP has native Linux support and requires the **KerbTrack mod** for head tracking:

1. Download and install the [KerbTrack mod](https://spacedock.info/mod/1745/KerbTrack)
2. In game settings, enable TrackIR if available
3. Access KerbTrack settings from the pause menu to configure head tracking
