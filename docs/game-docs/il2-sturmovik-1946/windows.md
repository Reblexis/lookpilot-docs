## LookPilot Configuration
1. Set protocol to `freetrack`
2. Click **Start** to begin tracking

## IL-2 Sturmovik: 1946 Setup
1. Head tracking should work automatically. If it doesn't respond, make sure you are in cockpit view (press F1).
2. It is recommended to unmap your joystick hat switches from "View" controls in the game's input settings to avoid conflicts with head tracking.
