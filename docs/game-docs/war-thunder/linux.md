## LookPilot Configuration
1. Set protocol to `virtual joystick`
2. Launch the game
3. Click **Start tracking** to begin tracking

## War Thunder Setup
1. In the game go to `Settings` ‚Üí `Controls`
2. Bind the following axes:
   - `View in battle - Air (X-axis)` ‚Üí Right Thumb Horizontal
   - `View in battle - Air (Y-axis)` ‚Üí Right Thumb Vertical
3. For both axes, set `Relative control` to `No`
4. For the Y-axis, set `Invert Axis` to `Yes`
