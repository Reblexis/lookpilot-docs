## LookPilot Configuration
1. Set protocol to `freetrack` (this emulates TrackIR - no TrackIR hardware is needed)
2. Click **Start** to begin tracking

## Aerofly FS 4 Setup
1. Launch the game
2. Enable the TrackIR / head tracking option in Aerofly FS 4's settings

## Fallback
If the freetrack route does not respond, the `virtual joystick` protocol is an alternative: select it in LookPilot, then bind the view axes in Aerofly FS 4's controls to the virtual joystick.
