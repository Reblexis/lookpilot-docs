Common questions about LookPilot's editions, purchases, and updates.

## Standalone vs Steam version

- There are **no significant functional differences** between the standalone app and the Steam version.
- **Owning either unlocks both.** Bought on Steam? Redeem the standalone unlock at lookpilot.app/redeem-steam. Bought the standalone? Get your Steam key at lookpilot.app/profile.
- The **Steam version requires the Steam client** to be running and logged in (LookPilot starts it automatically if needed). The standalone version does not.
- You do not need to uninstall one version to use the other.

## Updates

- **Steam**: updates arrive automatically through Steam.
- **Standalone on Windows**: auto-updates in the background.
- **Standalone on Linux (AppImage)**: the app notifies you about new versions, but you install them manually - all releases are at github.com/Reblexis/lookpilot-downloads/releases.

## Rolling back to an older version

- **Steam**: right-click LookPilot > Properties > Betas, and pick an older version.
- **Standalone**: download any release from github.com/Reblexis/lookpilot-downloads/releases.

## Purchase and library FAQ

- **LookPilot missing from your Steam library?** Steam's library filter shows Games only by default - open the filter (funnel icon) and tick "Software".
- **Library still shows "LookPilot Demo" after buying?** Harmless - the full app appears once the Software filter is enabled.
- **Is the Steam version restricted to Steam games?** No - it works with games from any store or launcher (GOG, standalone installers, etc.).
- **Do I need TrackIR hardware?** No - LookPilot emulates the TrackIR/FreeTrack protocols using only your webcam.
- **Does it work with glasses?** Head tracking is unaffected. Eye tracking is less accurate with glasses - consider disabling it.

## Drivers installed with the app

- The virtual gamepad driver (ViGEm Bus on Windows) is installed only if you enable virtual joystick output - not during normal installation.
